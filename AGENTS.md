# Agent Rules For Public Obsidian Vault

This vault is the public display layer for cleaned technical learning notes.

## Public Scope

Only publish:

- Mathematics
- Computer Science
- Communications

Do not add humanities, history, politics, social science reading notes, private reflections, WeRead raw exports, draft extractions, or long copyrighted excerpts.

## Source Of Truth

The private Obsidian vault is the source of truth. This public vault receives cleaned exports only.

## Before Editing

- Keep edits scoped to public-safe technical notes.
- Do not copy private-only links or raw source text into this vault.
- Preserve existing `.obsidian` client settings unless the user explicitly asks to change them.
